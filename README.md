# LOOP ENERGIA - GEV

Sistema otimizado para gestão da jornada do cliente Loop Energia.

## 🚀 Funcionalidades

- **Gestão de Carteira**: Visualização e edição de clientes.
- **Jornada do Cliente**: Acompanhamento de estágios (Timeline).
- **Gerador de Conteúdo**: Integração com Google Gemini para gerar mensagens personalizadas.
- **Módulo de Indicações**: Sistema de indicações integrado.
- **Dashboard**: Gráficos e métricas de desempenho.
- **Exportação**: Geração de documentos em PDF.

## 🛠️ Tecnologias Utilizadas

- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (Ícones)
- **Google Gemini API** (IA para geração de conteúdo)
- **Vite** (Build tool)

## 📦 Como rodar o projeto localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/loop-energia-gev.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure as variáveis de ambiente no arquivo `.env`:
   ```env
   GEMINI_API_KEY=sua_chave_aqui
   ```
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
